####################################
Fighter Pilot
####################################

This is the source codes for the game Figher Pilot which
is developed by Qt.

----------------------
Features
----------------------

- Game: scene
- Roles: Player, Bullet, Enemy
- Scores: Health, Score

----------------------
Prequisitions
----------------------

- ``Ubuntu 14.04``
- ``GCC``
- ``Qt5``

----------------------
How it runs
----------------------

::

    $ qmake
    $ make
    $ ./build/debug/FigherPilot
